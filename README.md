## 🚀 Sobre o projeto

Este repositório reúne uma coleção de **17 exercícios práticos desenvolvidos em Python** com foco no aprendizado e na aplicação dos principais conceitos de **Programação Orientada a Objetos (POO)**.

Os exercícios foram desenvolvidos progressivamente para praticar conceitos fundamentais da orientação a objetos, desde a criação de classes e objetos até conceitos mais avançados, como encapsulamento, herança, abstração, polimorfismo e uso de `property`, getters e setters.

O objetivo do repositório é consolidar conhecimentos de POO através da resolução de diferentes problemas e cenários práticos, desenvolvendo a capacidade de estruturar e organizar código utilizando os princípios da programação orientada a objetos.

Trata-se de um repositório de estudos desenvolvido durante minha jornada de aprendizado em **Python e Programação Orientada a Objetos**.

## 📂 Exercícios

O repositório contém **17 exercícios independentes**, desenvolvidos para praticar diferentes conceitos de Programação Orientada a Objetos em Python.

Cada exercício possui seu próprio objetivo e implementação, permitindo estudar os conceitos individualmente e observar como eles podem ser aplicados em diferentes situações.

## 🎯 Objetivos de aprendizado

* Compreender os fundamentos da Programação Orientada a Objetos
* Aprender a modelar problemas utilizando classes e objetos
* Praticar encapsulamento e controle de acesso aos atributos
* Utilizar getters, setters e `@property`
* Compreender o funcionamento da herança
* Aplicar polimorfismo em diferentes contextos
* Trabalhar com classes e métodos abstratos
* Aprender a validar dados dentro das classes
* Utilizar tratamento de exceções para controlar erros
* Desenvolver código mais organizado, reutilizável e estruturado
* Evoluir a capacidade de transformar problemas em modelos orientados a objetos
